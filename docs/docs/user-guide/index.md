# User Guide

This section walks you through every step of uploading a dataset.

| Step | What you do |
|------|-------------|
| [1 – Sign in](login.md) | Authenticate via Zenodo OAuth |
| [2 – Metadata](metadata.md) | Title, description, creators, keywords, license |
| [3 – Upload files](upload.md) | Drag & drop up to 100 files (≤ 50 GB total) |
| [4 – Publish](publish.md) | Review and submit to the MatWerk community |

!!! tip "Sandbox vs production"
    The application defaults to **sandbox.zenodo.org** so you can practise without
    creating real public records. Switch to production by setting
    `ZENODO_BASE_URL=https://zenodo.org` in your `.env` file.
